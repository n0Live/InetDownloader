InetDownloader - Delphi component
============

Компонент предназначен для получения  данных  из интернета в виде файла,
строки или потока. Также при помощи компонента  можно заливать файлы на
сервер. Работает с протоколами HTTP, HTTPS, FTP.
Подробное описание см. в файле InetDownloaderUnit.pas

Распространяется под лицензией GPLv3.
